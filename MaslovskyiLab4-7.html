Google Apps Glossary

Gmail
A free email service provided by Google that allows users to send and receive emails.
Google Drive
A cloud-based file storage service provided by Google that allows users to store and share files online.
Google Docs
A web-based word processing application provided by Google that allows users to create and edit documents online.
Google Sheets
A web-based spreadsheet application provided by Google that allows users to create and edit spreadsheets online.
Google Slides
A web-based presentation application provided by Google that allows users to create and edit slide presentations online.
Google Calendar
A web-based calendar application provided by Google that allows users to manage events and appointments online.
Google Meet
A video conferencing and online meeting platform provided by Google that allows users to host virtual meetings and collaborate in real-time.
Google Classroom
An online learning platform provided by Google that allows teachers to create and manage virtual classrooms for their students.
Google Photos
A cloud-based photo storage and sharing service provided by Google that allows users to store, organize, and share photos online.
Google Maps
A web-based mapping service provided by Google that allows users to view maps, get directions, and explore locations.
